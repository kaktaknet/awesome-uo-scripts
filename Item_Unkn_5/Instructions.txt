Sven's Course Plotting System - V1.0
This system allows boats to be guided by PINs on ANY normal sphere t_map.
The maps may be either player created or bought, and can be any size or centered at any location.

This ship system can also be added to Swindler's Boat Fix.


Instructions for installation and integration into Swindler's Boat Fix:

-Installing without any other boat modification scripts.
	Put Boats_MapGuidance_Main.scp and Boats_MapGuidance_Supporting.scp into your scripts directory.
	Restart Shard (Resyncs may cause problems with this script).

-Installing WITH "Swindler's Boat Fix" (Version Updated on 26-04-2002)
	Put Boats_MapGuidance_Main.scp and Boats_MapGuidance_Supporting.scp into your scripts directory.
	Make sure you have the latest version of Swindler's Boat Fix (scriptsharing.shorturl.com)
	Open shipfix.scp (default name for Swindler's boat fix).
	Search the file for [SPEECH spk_ship_cmds]
	Remove this entire section.
	Open Boats_MapGuidance_Supporting.scp and search for spk_ship_cmds.
	In spk_ship_cmds add "SHIPFIX" (without quotations) on the line directly under the following lines
	SHIPTURNLEFT
	SHIPTURNRIGHT
	SHIPTURN
