### Folder Naming Convention

All folders follow a consistent naming pattern:

```
Category_Type_v[version]
```

Examples:
- `House_Decoration_v1` through `House_Decoration_v13` - Various decoration scripts
- `Spell_Evocation_v1` through `Spell_Evocation_v8` - Evocation spell implementations
- `Item_Weapon_v1` through `Item_Weapon_v13` - Different weapon systems
- `NPC_Vendor_v1` through `NPC_Vendor_v4` - Vendor implementations
- `Quest_Event_v1` through `Quest_Event_v46` - Quest and event systems

Legacy folders for unclassified items:
- `House_Legacy_v1` through `House_Legacy_v23` - Unclassified housing scripts
- `NPC_Legacy_v1` through `NPC_Legacy_v4` - Unclassified NPC scripts
- `Chat_Legacy_v1` - Unclassified chat scripts
- `Item_Legacy_v1` - Unclassified item scripts
