## Installation Guide

### Quick Start

1. Browse the folders above
2. Choose the system that fits your needs
3. Copy the folder to your Sphere `scripts/` directory
4. Read the individual README.md file in that folder
5. Update sphere.ini if needed
6. Restart your Sphere server

### Recommended Approach

1. Start with one or two core systems
2. Test thoroughly in a development environment
3. Gradually add more systems as needed
4. Monitor server performance and stability

## Tips & Best Practices

- **Read READMEs**: Each folder has a README.md with specific installation instructions
- **Check Dependencies**: Some systems may depend on others - read the notes
- **Test First**: Always test new systems in a dev environment
- **Backup**: Back up your scripts directory before making changes
- **Version Control**: Keep track of which systems you're using
- **Community**: Check the awesome-uo-scripts GitHub for community feedback
