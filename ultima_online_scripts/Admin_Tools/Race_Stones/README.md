# Race_Stones

## Description
Race change stones for character customization

## Category
Admin Tools

## Sphere Compatibility
- Works with most Sphere versions
- Primary target: Sphere 0.55i

## Files
- `aquarian_race_stone.txt`
- `daemon_race_stone.txt`
- `elf_race_stone.txt`
- `human_race_stone.txt`
- `lost_spirit_race_stone.txt`
- `orc_race_stone.txt`
- `undead_race_stone.txt`

## Installation
1. Copy this folder to your Sphere `scripts/` directory
2. Review the script files for any configuration needed
3. Add to your sphere.ini if required
4. Restart the server

## Usage
See individual script files for usage instructions and configuration options.

## Notes
- This is an admin/GM tool
- Requires appropriate permissions to use
