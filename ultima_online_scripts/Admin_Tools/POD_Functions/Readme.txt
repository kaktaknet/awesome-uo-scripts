---===INTRODUCTION===---
	This package includes the basic files
require for most OnlineUO scripts.
	Two files are included in the package:

	-#PODFunctions.scp
	:All Basic Functions for scripting
	:See documentations inside the script

	-#PODEvents.scp
	:NPC AI and events.

---===Installation===---
	Place the two files in your scripts
directory. #Drive:\...\SPHERE\Scripts\ 
by default.
	Reboot sphere.
