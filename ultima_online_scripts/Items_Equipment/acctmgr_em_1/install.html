System requirements

  • A web server (tested with Apache 1.3)
  • PHP 4 or better
  • Access to the sphere directory

Installation/Setup
To install, simply edit the configuration file to fit your needs and upload all the included files (minus this one if you desire) into a web-accessible directory.  Configuration is achieved through changing the different variables to match your setup.  The variables should be straight forward.  Near the end there is a section of functions that define the output of the program for initial startup and success message.  The ones that come with the default program are pretty basic and should be self explanatory, but only experienced PHP users should attempt to modify them further.
